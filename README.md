# @hitv/ua

> Is a simple userAgent parser for develop kit.

## Installation

```sh
npm i @hitv/ua
```

## Usage

```js
import ua from '@hitv/ua'

console.log(ua)

// =>
{
  "safari": true,
  "version": {
    "major": 9,
    "string": "601.1.46",
    "minor": 0
  },
  "ipad": true,
  "webkit": true,
  "name": "safari",
  "platform": "ipad"
}
```
